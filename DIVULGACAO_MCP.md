### Opção 2: Cursor / Windsurf / Claude
Adicione no arquivo de configuração:

```json

    "zanly-tech": {
      "command": "npx",
      "args": ["-y", "@smithery/cli", "run", "victor-zanardo-wy1b/zanly-tech"]
    }
  }
}
```
